HODL Totals is a do-it-yourself coin and NFT tracking system based on Google Sheets. Follow the steps below to gather your transaction data into a consistent format, reconcile the sheet data with your holdings and then calculate your gains and losses.

These sheets are intended to be sufficient as an audit trail to back up your US crypto tax filing. This data can be used to generate yearly Income Reports and a 1099 Data Reports (for Form 8453).

    • To get started, create a new coin tracking sheet using the Track New menu in the Extensions » HODL Totals menu bar.
    • Do not enter data into the gray columns of any HODL Totals sheet. This content is always calculated for you.
    • Each transaction for a given coin should have a unique row in that coin's tracking sheet.
    • Enter coin inflows as positive numbers, coin outflows as negative numbers in the Net Change column.
    • If some amount of coin entered your control, enter that coin amount in the Acquired column.
    • If some amount of coin left your control, enter that coin amount in the Disposed column.
    • If you know the value of the coin amount at the time of the transaction, set the Valuation Strategy to Value Known
      and record the US Dollar value of that amount in the neighboring Value (USD) column.
    • When transferring coins between addresses you control, set Category to Transfer In/Out and Strategy to n/a.
    • Non-Fungible Tokens are tracked via the specific ID method. That's what makes then non-fungible.
    • Each NFT tracking sheet corresponds to an address that can hold many NFTs over time.
    • Create a unique row for each NFT you interact with, adding name and identification information in the frozen columns.
    • NFT Inflow information is recorded on the left side of each row, including many details that could span many transactions.
    • NFT Outflow information is recorded on the right side of each row, including many details that could span many transactions.
    • Each NFT Outflow must match up with an NFT Inflow on the same row.
    • Track each NFT's cost basis accurately by accounting for the USD value of all crypto transactions involving each NFT
      by copying the USD values from your coin sheets and pasting them into the NFT sheet's (USD) columns.
    • For NFT sheets, the Calculate gain/loss command assigns tax treatments to both the Inflow Status and Outflow Status columns.
    • For each transaction, provide Wallet/Account information and a Transaction ID if possible.
    • Use the Tx ✔ column to track your progress while updating or reconciling your transactions.
    • Reset the HODL Totals sheet at any time to refresh a summary of your holdings across all coins and NFTs.
    • Enter your holdings on the HODL Totals sheet periodically to reconcile your records with reality.
    • If the USD amount paid to receive the coin inflow is known, enter in the Inflow Value (USD) column and set strategy to Value Known.
    • If the USD amount received for the coin outflow is known, enter in the Outflow Value (USD) column and set strategy to Value Known.
    • If the coin price in USD at the time of the transaction is known, enter in the Fair Mkt Value Price column and set strategy to Price Known.
    • Enter data for Coin Highs/Lows and set strategy to Avg Daily Price Variation to track value of coins with very low trading volume and
      copy/paste Coin High/Low data from the historical values listed on https://coinmarketcap.com for that coin.
    • Transactions must be in chronological order before a calculation will succeed; do this by sorting the Date & Time column into A→Z order.
    • The calculated Status column will provide useful taxation information for each transaction.
    • Taxable or Not Taxable status will be assigned based on the categories you configure. Edit the hidden Category sheets to customize them.
    • If a coin transaction requires spending some coin amounts held for more than a year and some coin amounts held less than a year, then
      the transaction will get split into two rows: one tracking the Long-term Gain/Loss and the other tracking the Short-term Gain/Loss.
    • Use the Tax Doc Link column to store links to other sheets or yearly tax reports that list each transaction's taxable gain or loss.
